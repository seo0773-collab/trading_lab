# yoon1c(섹터 레짐 필터) vs yoon1/yoon1b

yoon1=SPY 단일 시장필터, yoon1b=+게인 1.25, yoon1c=게인 1.25 + **하이브리드**(SPY 시장필터 ∧ 종목별 섹터필터 동시; 둘 다 약세면 0.5×0.5=0.25 이중 축소). 공통: top_k 20·monthly·floor 1.0. 주 벤치=SPY.

## holdout(test) 판정
- yoon1b(SPY필터): CAGR 19.2% · MDD -22.9% · Sharpe 1.335 · 노출 90.8%
- yoon1c(섹터필터): CAGR 18.0% · MDD -19.6% · Sharpe 1.340 · 노출 86.1%
- 차이(yoon1c−yoon1b): Sharpe 0.006 · CAGR -1.3%

→ **섹터 레짐 필터 우위**: 위험조정 개선.

## 해석 (수동 정리)

- **순수 섹터 필터(SPY OFF)는 실패**했었다(이전 실행: test Sharpe 1.286·MDD -25.6%).
  섹터 신호가 덜 동기화돼 위기 때 노출을 더 떠안아 낙폭이 커졌다.
- **하이브리드(SPY ∧ 섹터, 둘 다 약세면 0.25로 이중 축소)는 성공**. 두 게이트가
  상호 보완: SPY가 전면 약세를 빠르게 잡고, 섹터가 개별 약세 종목을 추가로 덜어낸다.
- holdout test에서 yoon1c가 **세 변형 중 Sharpe 최고(1.340)·MDD 최저(-19.6%)**.
  all 구간에서도 Sharpe 최고(1.151). **두 구간 모두 일관된 개선 = 노이즈 아님.**
- 단 Sharpe 우위는 yoon1b 대비 미미(+0.006). **실질 차이는 방어력**: MDD가 yoon1b
  -22.9%→-19.6%로 3.3%p 개선되는 대신 CAGR -1.3%p(19.2→18.0%) 양보. 노출도 91→86%로
  더 보수적. **'더 방어적이고 위험조정 동급~소폭↑' 변형**으로 자리매김.

**결론**: yoon1b=수익 우선, yoon1c(하이브리드)=방어 우선(최저 낙폭·최고 Sharpe).
용도가 갈리는 두 후보로 공존 가치 있음.
