# 전략 명세: yoon1k (계층 포트폴리오 — 원/달러 × 통화별 yoon1j)

> 두 개의 [[yoon1j]] 계열 전략을 통화별로 운용하고, 최상위에서 원/달러 비중을
> 주기적으로 리밸런싱하는 **2단계 fund-of-funds**. 하위는 기존
> `ProfilePortfolioHandler` 를 위임 호출하고, 이 전략(`Yoon1kHandler`)은 **환율 환산 +
> 통화 리밸런싱 결합**만 담당한다(공통 인프라 무수정).

## 1. 식별 정보

| 항목 | 값 |
| --- | --- |
| strategy_id | yoon1k |
| 구조 | 원/달러 50/50 월간 리밸런싱 × {미국 yoon1j(USD), 한국 yoon1j_kr(KRW)} |
| config | configs/strategies/yoon1k.json |
| 핸들러 | src/trading_lab/strategies/yoon1k.py (`Yoon1kHandler`) |
| 작성일 | 2026-06-25 |
| 상태 | validation (원화기준 2007~2026 실데이터 검증, 121+ 테스트 통과) |
| 결과 단위 | 1 run = 1 통합 포트폴리오(원화 기준 NAV) |

## 2. 구조

```
[최상위] 원/달러 통화 배분 리밸런싱 (50/50, 월간)
   ├─ 원화 슬리브 → 한국 yoon1j_kr (KODEX 섹터5 + 국고채10년/골드)
   └─ 달러 슬리브 → 미국 yoon1j   (SPDR 섹터11 + TLT/GLD), USD→KRW 환율 환산
```

- **기준 통화 = 원화(KRW)**. 미국 슬리브 NAV에 USD/KRW(`KRW=X`) 환율을 곱해 원화로
  환산한다 → 환율 변동이 수익에 반영(통화 다변화).
- 최상위는 두 슬리브를 목표비중(50/50)으로 **매월 되돌리는** 단순 리밸런싱.
- 하위 yoon1j는 각자 `phase="all"` 로 전체 NAV를 만들고, 메타가 결합 후 phase 윈도우를
  슬라이스한다(train 0.6 / val 0.2 / test 0.2).

## 3. 핵심 아이디어 — 무상관 결합의 분산 효과

한국 yoon1j_kr와 미국 yoon1j(원화 환산)의 **일간수익률 상관 ≈ -0.026**(거의 완전
무상관). 두 무상관 방어형 전략을 50/50으로 묶으면 변동성이 ~√2 줄고, 수익은 평균인데
위험만 급감 → **위험조정 성과(Sharpe)가 개별 슬리브보다 도약**한다. 환율 다변화가
위기 시 자연 헤지(원화 약세 → 미국 슬리브 원화가치↑)를 더한다.

## 4. 검증 결과 (원화 기준, 2007~2026, 16.2년)

| 라인 | CAGR | MDD | Sharpe |
| --- | ---: | ---: | ---: |
| **yoon1k (통합 50/50)** | 14.2% | **-14.4%** | **1.151** |
| 벤치 50/50 (KODEX200+SPY원화, 전략없음) | 17.0% | -31.4% | 1.104 |
| 미국 yoon1j (원화환산) | 15.2% | -16.3% | 0.856 |
| 한국 yoon1j_kr (원화) | 12.1% | -30.9% | 0.772 |
| SPY (원화환산) b&h | 16.8% | -30.2% | 0.843 |

- **Sharpe·MDD 모두 모든 라인 중 최고** — 1+1이 2를 넘었다(개별 0.77~0.86 → 1.151).
- **전략 부가가치 확인**: 통합(1.151) > 같은 통화배분의 전략없는 벤치(1.104). yoon1j
  방어가 MDD를 -31.4%→-14.4%로 절반화.
- **한계**: 절대수익은 양보(가장 부드럽지만 가장 적게 범 = 방어형), 기간 16년(한국 ETF
  제약), 통화 리밸런싱 환전비용 미반영. 짧은 강세 holdout(test 2022~)에선 벤치가 앞섬.

## 5. 판정

위험조정·낙폭 우선이면 강력한 선택(세션 검증 중 최고 Sharpe·최저 MDD). 절대수익
극대화가 목적이면 부적합. **성격 = 무상관 국가/통화 다변화 + 방어형 yoon1j의 결합.**

오픈 이슈: ① 통화 비율 스윕(50/50 vs 60/40 등) ② 통화 배분 신호(환율 추세·슬리브 강도)
③ 3국 이상 확장(일본·신흥국) ④ 통화 리밸런싱 환전비용 모델링.
