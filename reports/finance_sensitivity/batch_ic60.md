# fin-sensitivity-v1 — 배치 IC60 + placebo 순열검정 (2026-06-14)

유니버스 34종목(섹터 분산), EDGAR 분기재무, 타깃=SPY 대비 60일 초과수익,
피처=재설계(YoY 변화+서프라이즈+밸류 상호작용). 예측 vs 실제의 Spearman IC.

## 판정

| 지표 | 값 |
| --- | --- |
| 사용 종목 | 34 (no_data 0) |
| **real 평균 IC60** | **+0.0124** (양수 53%, 18/34, 중앙값 +0.028) |
| placebo 평균 IC60 | -0.0176 ± 0.0222 (n=120 순열) |
| 순열검정 p값 | **0.091** |
| **판정** | **약한/모호 — placebo 대비 경계. 단일종목(A) 엣지 불충분, B 횡단면 풀링 검토** |

- 명세 §9 합격선(IC가 placebo 대비 95% 유의)에는 **미달**(p=0.091 > 0.05).
- 그러나 §2 무효화 조건(IC ≤ 0)에도 해당하지 않음 — real이 placebo보다 +0.030 높음.
  즉 순수 노이즈는 아니나, 단일 종목 시계열(종목당 ~50 표본)로는 신호가 약하다.
- placebo 평균이 0이 아니라 -0.018인 점에 주의: rolling Ridge가 셔플 피처에서 약한
  음의 편향을 가짐(소표본). 그래서 비교 기준은 0이 아니라 placebo 분포다.
